# Versioning Agreement

- Current version is `1.0.0-alpha`.
- Stay on `alpha` type/stage until the user explicitly says otherwise.
- Before changing the version for any reason, check in with the user and get confirmation first.
- Version increment rules (major.minor.patch-type):
  - Minor changes -> increment patch.
  - Updates -> increment minor and reset patch to 0.
  - Major updates -> increment major and reset minor/patch to 0.
