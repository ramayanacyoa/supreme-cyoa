Latest Update

Update 1.0.5 — Family Setup Toggle + Story Name Color Refresh

Added an Enable Family Setup button that reveals or hides family fields, and switches to Disable Family Setup when active.

Disabling Family Setup during a run immediately returns story names to canon defaults; re-enabling later restores your previously saved custom family names.

If you try enabling Family Setup for the first time after a run has already started, you now see: Not Available. Finish current game session.

Story name highlights now use a softer gold that better matches the story card styling, and the quest start button has extra spacing below the player name input.
