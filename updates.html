Latest Update

Update 1.0.3 — Live Systems Integration

Integrated new systems directly into the live adventure UI with save export/import and an in-session Dharma Console.

What’s included:

  • Expanded evolving trait model (dharma, aggression, compassion, strategy, honor) that modifies combat and ending outcomes.
  • Dynamic companion relationship threshold states with loyalty pacts, betrayal risk, and combat synergy hooks.
  • JSON export/import save architecture for full game-state persistence across sessions and devices.
  • Game Master-style Dharma Sandbox controls for branch simulation, stat tuning, event forcing, and content unlock testing.
