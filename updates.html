Update 1.0.13 — Stabilized Modular RPG Interface

Stabilized scene routing and expanded the class-based adventure with modular systems, a polished three-panel RPG HUD, dynamic image fallbacks, autosave, day/night progression, Slumberland rest scenes, and 40+ connected side scenes.

What’s included:

  • Fixed scene generation and transition stability by removing render-time scene mutation.
  • Added modular inventory, quest, party, kingdom, save/load, and time systems under dedicated folders.
  • Added a responsive dark-gold game interface with top status bar, side panels, fade animations, and image overlays.
  • Added Slumberland rest routing plus 42 new connected expansion scenes across Ayodhya, Exile, Forest, Kishkindha, Lanka, and Return.

Added unique opening scenes for Lakshmana, Bharata, Ravana, and Sita so each selected hero begins from their own narrative viewpoint.

What’s included:

  • New hero-specific scene routing from the opening scene.
  • New introductory scenes centered on each hero’s role and motivations.
  • All new paths reconnect into the existing campaign timeline for continued branching.

Update 1.0.10 — Playable Hero Selection

You can now start the adventure as Rama (classic), Lakshmana, Bharata, Ravana, or Sita from the opening setup card.

What’s included:

  • New hero selector with five playable role options before starting the run.
  • Each hero now loads a lore-aligned family cast mapping for dynamic story placeholders.
  • Name field is now optional and falls back to the selected hero’s canonical name.

Update 1.0.9 — Persistent World, Account Saves, Settings, and Slumberland Expansion

Added realtime world simulation, account management, JSON save workflow, settings controls, sleep/dream progression, and persistent travel encounters while removing AI continuation.

What’s included:

  • New account setup flow and save snapshot support through downloadable JSON files.
  • New settings panel for realtime progression, travel encounters, and dream realm behavior.
  • Added Wisdom and Knowledge player stats with Slumberland dream boosts and travel encounter growth.
  • Removed AI continuation feature and shifted narrative growth to deterministic world simulation.

Update 1.0.6 — Expanded RPG Campaign, Stat Modal, and Day/Night Story Cycle

Added a major campaign expansion with long-form arc progression through Lanka War, Return to Ayodhya, and Uttara Kanda, along with new player stat systems and modal UI.

What’s included:

  • Added six combat attributes: Strength, Defense, Speed, Agility, Stamina, and Endurance.
  • Added a dedicated Stats modal accessible from the in-story toolbar.
  • Added world clock progression with day/night cadence across extended scene chains.
  • Replaced the previous Lanka placeholder with detailed multi-scene progression from war planning to multiple endings.

    • Update 1.0.7 — Storyline Flowchart Fix

    Fully repaired the My Storyline experience so the flowchart and timeline reflect actual choices and special branch outcomes.

    What’s included:

    • Timeline list now shows step-by-step transitions with each chosen option and destination scene.
    • Flowchart now includes special transition branches (random/conditional outcomes), not just direct positive scene IDs.
    • Improved reveal toggle wording: “Hide Unexplored Scenes”.
    • Bumped hidden console update marker to Update 1.0.7 for release tracking.

      • Update 1.0.8 — Day/Night Mode Toggle (In-Game vs Real-Life)

      You can now choose whether day/night behavior follows in-game rules or your actual local real-world time.

      What’s included:

      • New toolbar toggle to switch between In-Game Day/Night and Real-Life Day/Night.
      • In Real-Life mode, random/luck-based scene outcomes are now influenced by actual time of day.
      • Daytime grants better luck odds; nighttime lowers luck odds, while player choices still drive progression.
      • The selected mode is saved in local storage and restored on page load.
