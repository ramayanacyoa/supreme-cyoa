# Features

_Last updated: 2026-04-18 (UTC)_

## Gameplay and UI Features

- Branching Ramayana CYOA narrative with scene-based progression and route-dependent outcomes.
- Undo support for the most recent choice.
- Restart options from ending scenes.
- My Storyline modal that lists visited scenes and includes each scene description.
- Hover/focus helper text on art buttons:
  - Start button: `begin your quest`
  - Restart button: `restart`
  - Undo button: `undo`
- Responsive navigation and secondary lore pages.
