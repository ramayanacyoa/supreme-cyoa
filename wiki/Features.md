# Features

_Last updated: 2026-04-24 (UTC)_

## Gameplay and UI Features

- Branching Ramayana CYOA narrative with scene-based progression and route-dependent outcomes.
- Name-based story personalization (`{{name}}` interpolation).
- Expanded family personalization inputs (player, father, mothers, spouse, and three siblings) with fallback canon names.
- Sibling gender toggle support with dynamic pronoun interpolation in scenes.
- Undo support for recent choices.
- Restart options from ending scenes.
- Storyline modal with visited-scene summaries.
- Training Ground trivia sequence after meeting Hanuman.
- Farthest current progression end card titled **The Ramayana Adventure: Lanka** with **Coming Soon**.
- Added royal-court branch where a second argument against exile can trigger an execution ending.
- Character names highlighted in bright teal across scene content and timeline display.

## Navigation and Audio

- Unified top navbar style across index, updates, prequel, and people pages.
- Native soundtrack player in navbar using only the built-in browser controls (no extra custom slider).
- Soundtrack attempts autoplay on page entry.
- Track position, pause/play state, mute state, and volume persist across page navigation for seamless playback.
- Stored soundtrack volume preference in local storage.
- Starting the quest sets soundtrack volume to **50%**.

## People Page Coverage

- Includes all requested key characters from the Ramayana character set.
- Includes only these kingdoms under grouped entries: **Ayodhya**, **Lanka**, **Mithila**, and **Kishkindha**.
- Every people/kingdom card includes a Wikipedia link button.
